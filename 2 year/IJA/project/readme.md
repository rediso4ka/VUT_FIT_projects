## Requirements

- Java 17 (JDK 17)
- Maven >= 3.6.3 (not tested on older versions)

## Compilation

```bash
mvn clean
mvn compile
mvn package
```

## Running

Running from the root of the project is mandatory, otherwise the game might not be able to find the resources.

```bash
java -jar ./target/pacman-1.0.0.jar 
```
